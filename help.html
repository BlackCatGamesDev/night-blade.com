Help Wanted

Making a high quality game like Nightblade is a lot of work and we're in need of some skilled people to help out in key areas. We're going to post up as much information as we can about each task that we need help with, so in some cases there will be enough information here for you to start helping right now. In other cases, you'll have to get into touch with us for more information or for files to work with.

Take a look below and help us make Nightblade into a masterpiece!

3D Modeller

We're using a number of old Thievery models, for loot, household items, decorations, furniture and so on. These objects are badly in need of an update.

Download the latest Nightblade Alpha and have a look around. If you see any meshes you think you can improve upon, then contact us and let us know!

-:-

Texture Artist

We're also using a number of old textures. Most of these are just plain 256x256 textures. It'd be great if some of the more common ones were replaced with better quality textures, complete with normal maps and specular where appropriate. If you're interested in helping with this, contact us and we can provide the original textures to work from.

-:-

Mapping

At the moment we are using maps which have been converted directly from Thievery (and thus from the original UT). It would be great to get some new maps created making use of improved content and modern mapping techniques. If you are interested it is possible to start mapping using the latest Nightblade Alpha. The Mapper's Guides should help you get started.

If you need any further help feel free to contact us or ask questions on the forums.
